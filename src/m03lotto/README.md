    (1) 1 ~ 45 까지 랜덤으로 번호 6개 + 보너스 번호 1개 를 가진다.
    (2) 사용자에게 번호를 받아 확인, 결과를 출력함
    (3) 룰 
        6개 번호 일치 : 1등
        5개 번호 일치 + 보너스 1개 번호 일치 : 2등
        5개 번호 일치 : 3등
        4개 번호 일치 : 4등
        3개 번호 일치 : 5등
